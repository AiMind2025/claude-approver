# 📱 Claude 手机审批 - 使用说明

> 让你在手机上批准 Claude Code 的操作，不用守着电脑。

---

## 🚀 三步开始用

### 第一步：启动服务器

双击 `start.bat`，看到下面这样就是成功了：

```
╔══════════════════════════════════════════════════════╗
║          🤖 Claude 手机审批服务器 (增强版)         ║
╚══════════════════════════════════════════════════════╝

✅ HTTP 服务: http://localhost:8765
[ngrok] ✅ 公网地址: https://xxxx.ngrok-free.dev

─── 访问地址 ───────────────────────────
  本地: http://localhost:8765
  公网: https://xxxx.ngrok-free.dev?token=你的密码
```

> ⚠️ **窗口别关**，关了服务器就停了

---

### 第二步：打开网页，设个密码

电脑浏览器打开：**http://localhost:8765**

第一次用会要你设密码，随便输一个（至少 4 位），记住它。

---

### 第三步：手机打开

看命令行窗口里的 **公网地址** 那一行，复制到手机浏览器打开：

```
公网: https://xxxx.ngrok-free.dev?token=你的密码
```

> 💡 第一次打开会有个 ngrok 的安全提示页，点 **"Visit Site"** 就行

现在手机和电脑都能看到审批页面了。

---

## ✅ 怎么用

### 场景：Claude 要执行危险命令

1. Claude 发送审批请求（自动的，不用你管）
2. 你手机收到通知（如果配了微信推送）或者打开页面看
3. 看到待审批的命令，点 **✓ 批准** 或 **✗ 拒绝**
4. Claude 收到结果，继续或中止

### 页面说明

- **⏳ 待审批** — 等你决定的请求
- **📋 历史** — 已经批准/拒绝的（最近 50 条）
- 顶部红点数字 = 还有几个没处理

## 📲 微信推送（可选）

想让审批通知发到微信？

1. 去 https://sct.ftqq.com 注册，拿到 **SendKey**
2. 打开 `config.env`，找到这行：
   ```
   # SERVERCHAN_KEY=SCT362965TNnuteeyQr7P4xWsw4C8Cz6om
   ```
3. 去掉前面的 `#`，换成你的 SendKey：
   ```
   SERVERCHAN_KEY=你的SendKey
   ```
4. 重启服务器（关窗口，再双击 start.bat）

之后每次有审批请求，微信都会收到消息，点消息里的链接直接审批。

## ❓ 常见问题

**Q: 密码忘了怎么办？**
删掉 `.data` 文件夹里的 `auth.json`，重启服务器，重新设。

**Q: 手机打不开公网地址？**
- 确认命令行窗口没关
- 确认能看到 `[ngrok] ✅ 公网地址: ...`
- ngrok 链接第一次打开要点 "Visit Site"

**Q: 每次重启公网地址都变？**
正常，免费版 ngrok 就这样。想固定地址要去 ngrok 官网配置。

**Q: 不想用 ngrok，只在本机用？**
打开 `config.env`，把 `TUNNEL=ngrok` 改成 `TUNNEL=none`，重启。

## 🔧 Claude 里怎么用

在 Claude Code 里执行危险命令前，先调用审批脚本：

```bash
bash D:/projects/claude-approver/approve.sh "要执行的命令" "说明" "danger"
```

- 返回 0 = 你批准了，继续执行
- 返回 1 = 你拒绝了，中止
- 返回 2 = 等超时了（默认 5 分钟）
