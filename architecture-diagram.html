Claude 手机审批服务器 Architecture

AI 编程助手远程审批系统 — Claude Code / OpenCode 通过 MCP 协议将操作推送到手机审批

MCP 工具接口

  • • request_approval — 操作审批 (阻塞)
  • • ask_question — 提问等回复 (支持多轮)
  • • check_status — 非阻塞查询状态
  • • close_conversation — 结束对话
  • • get_server_info — 获取服务信息

推送通道

  • • 喵提醒 — 微信推送 (免费 100 条/天)
  • • SMTP 邮件 — 可选 (纯 Node.js 零依赖)
  • • ngrok 隧道 — 公网 HTTPS 访问
  • • DISABLE_PUSH=true — 禁用推送模式
  • • 推送内容含审批链接，手机直接操作

自愈机制

  • • 端口冲突 EADDRINUSE → 自动重启
  • • 每 60 秒健康自检 (/api/health)
  • • uncaughtException 全局捕获
  • • 连续崩溃 >5 次暂停，避免死循环
  • • killPortOccupant — 启动前清理旧进程
