## 测试 2：审批用例

**触发方式**：让 Claude 执行 `npm install`

**操作步骤**：
1. 在终端输入：`帮我安装依赖`
2. Claude 调用 `request_approval`，risk="normal"
3. 手机收到审批请求
4. 点击"批准"

**预期结果**：
- ✅ 手机收到推送通知
- ✅ 审批页显示风险标识
- ✅ 点击批准后 Claude 继续执行

**截图**：

![npm依赖审批](screenshots/npm依赖审批.jpg)

**测试结果**：✅ 通过
